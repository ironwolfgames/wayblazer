# Wayblazer Notes

## World Generation

Here are some common settings for world generation. All of the configurations below have the following settings:

- Width & Height : `256`
- Enable Ocean Continent Generation : `TRUE`
- Edges Are Ocean : `TRUE`
- WorldMapNoiseConfig:
  - Frequency: `0.005`
  - Octaves: `6`

### Typical World With A Few Continents

- Ocean Continent Noise Map Threshold : `0.52`
- Ocean Continent Noise Map Frequency: `0.01`

### World With One Main Continent

- Ocean Continent Noise Map Threshold : `0.35`
- Ocean Continent Noise Map Frequency: `0.008`

### World With Lots Of Islands

- Ocean Continent Noise Map Threshold : `0.55`
- Ocean Continent Noise Map Frequency: `0.024`
